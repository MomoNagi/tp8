# TP8

Ce dépôt contient l'implémentation du Travail Pratique 8 (tp8) du cours d'Apprentissage Automatique à Grande Échelle. 

Il propose une extension Python développée en C++ (via `pybind11`) pour calculer la médiane cumulative sur des tableaux de données.

Le projet est hébergé sur GitHub pour permettre la génération et l'hébergement de la documentation via Read the Docs.
